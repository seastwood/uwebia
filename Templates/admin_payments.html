Stripe Checkout is used for online purchases on your store. Customers pay on Stripe's hosted page and are redirected back to your site when the payment completes.

Stripe connection

Get keys at dashboard.stripe.com/apikeys. Use test keys (sk_test_..., pk_test_...) while you're setting things up; switch to live keys when you're ready to take real money.

How to set up the webhook ↓
  1. In Stripe Dashboard go to Developers → Webhooks → Add endpoint.
  2. Set the endpoint URL to:
    {{ webhook_url }}
  3. Listen for these events: checkout.session.completed, checkout.session.expired, payment_intent.payment_failed, charge.refunded.
  4. After saving the endpoint, copy the Signing secret (whsec_...) into the field below.
How to set up Stripe Tax ↓
  1. In Stripe Dashboard, go to Tax and click Get started. Fill in your business's origin address (where your business is physically located) and confirm.
  2. Add tax registrations in Tax → Registrations for every jurisdiction (US state, country) where you've registered to collect tax. Stripe only collects in jurisdictions you've added here — you must register with each state / country first.
  3. Stripe tracks economic-nexus thresholds automatically. When you approach a state's threshold, the Monitoring page flags it so you know to register.
  4. Set a Tax category on each of your products (in the product editor). Defaults to "General — physical goods" if unset.
  5. Check the box above and click Save. From now on every Stripe Checkout Session asks the buyer for a billing address and adds the right tax to the total.

Heads up: Stripe Tax bills $0.50 per transaction or 0.5% of order value, whichever is lower. See stripe.com/tax/pricing. You are still responsible for registering with each state / country and filing returns; Stripe Tax collects and reports, but doesn't file for you.
