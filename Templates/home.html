Features of uwebia:

  • An Admin page which strives for simplicity in allowing the creator to edit every aspect of their website.
  • Ability to upload images along with drag and drop functionality to sort and delete them.
  • Immediate toggling on and off of the public page.
  • Optional contact form.
  • Optional map pin showing set coordinates.

Future features of uwebia include:

  • Ability to simply create an account, log in, and start creating your website.
  • Ability to build multiple websites projects which can have multiple pages.
  • Ability to add, selectively place, and delete each section on the page.
  • Index menu for each section.
  • Shopfront section in which the user can advertise and sell things like products, tickets, and services if desired.
  • AI assistance.
  • Option to have your own user base and forum with different styles such as classic and modern.
  • Domain name registration option.
  • More improved UI
